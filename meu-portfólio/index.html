Skills

HTML é uma linguagem de marcação usada para criar e estruturar páginas da web. Ele utiliza tags para definir elementos como títulos, parágrafos, imagens e links, sendo interpretado por navegadores para exibir o conteúdo aos usuários.

CSS, ou Cascading Style Sheets, é uma linguagem usada para definir a aparência e o layout de páginas web. Ele permite estilizar elementos HTML, controlando aspectos como cores, fontes, espaçamentos, alinhamentos e outros atributos visuais.

JavaScript (JS) é uma linguagem de programação usada para adicionar interatividade e dinamismo às páginas da web. Permite criar animações, manipular elementos da página, fazer chamadas assíncronas e interagir com o DOM. JS é executado no navegador do usuário, tornando as páginas mais responsivas e funcionais.

WordPress (WP) é uma plataforma de gerenciamento de conteúdo (CMS) usada para criar e gerenciar sites e blogs. É flexível e personalizável com temas e plugins, permitindo que os usuários criem diversos tipos de sites.
